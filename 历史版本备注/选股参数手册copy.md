# A股选股参数手册：「主力洗盘完成·即将启动」

> 使用方法：分析任何一只股票时，对照本手册逐项检查，判断是否符合洗盘末期特征。
> 可以直接问："XX股票符合手册中的哪些条件？不符合哪些？后期怎样走会更符合？"


---


## 第一关：前置过滤条件

先用这些硬性条件筛出「有主力参与」的股票池，不满足的直接排除。

| 序号 | 条件 | 参数 | 为什么 |
|:---:|------|------|--------|
| F1 | 流通市值 | 30亿~300亿 | 太小易被操控、太大拉不动 |
| F2 | 近60日出现过涨停 | 至少1次日涨幅≥9.5% | 说明有资金关注过，不是僵尸股 |
| F3 | 近60日涨幅 | 10%~60% | 涨过一波（主力介入），但没涨太多（还有空间） |
| F4 | 近60日最高点回撤 | 从高点回落15%~30% | 正处于洗盘或洗盘接近尾声 |
| F5 | 股价与MA60关系 | 现价 > MA60 | 中期趋势向上，不碰下降通道的股票 |

**判断示例**：
- ✅ 某股60日内有过涨停，之后从高点回落了22%，当前价仍在MA60上方 → 通过
- ❌ 某股60日跌了15%，没涨停过，现价在MA60下方 → 排除

## 第二关：核心指标逐项检查

通过第一关后，对照以下7大核心指标逐一打分。


### 指标①：缩量程度 ⭐⭐⭐⭐⭐（权重最高）

> **核心逻辑**：主力洗盘的目的是把浮筹洗出去。当成交量萎缩到极致，说明该割的都割了，
> 剩下的都是坚定持有者或主力锁仓筹码。此时是启动前最后的窗口。

| 子项 | 达标参数 | 如何查看 |
|------|----------|----------|
| 5日/20日量比 | **≤ 0.6** | 近5日均量 ÷ 近20日均量 |
| 5日/60日量比 | **≤ 0.5** | 近5日均量 ÷ 近60日均量 |
| 近5日日均换手率 | **≤ 2%**（小盘股≤3%） | 看行情软件换手率 |
| 连续缩量 | 近3日成交量**逐日递减** | 今日量 < 昨日量 < 前日量 |
| 地量信号 | 当日成交量 ≈ 近60日最低量（±20%） | 和历史最低量对比 |

**达标判断**：
- 5项中满足3项以上 → ✅ 缩量达标
- 满足1~2项 → ⚠️ 部分达标，继续观察
- 均不满足 → ❌ 未缩量，不符合

**实战对照**：
```
例：某股近5日均量120万手，近20日均量280万手
    量比 = 120/280 = 0.43 ≤ 0.6 ✅
    换手率近5日平均0.6% ≤ 2% ✅
    最近3天量：135万→118万→108万，逐日递减 ✅
    → 缩量达标 ✅
```


### 指标②：横盘整理 ⭐⭐⭐⭐

> **核心逻辑**：洗盘末期股价不再大涨大跌，在窄幅区间内横向运行，
> 横盘时间越长说明洗盘越充分，后续爆发力越强。

| 子项 | 达标参数 | 说明 |
|------|----------|------|
| 价格波动幅度 | 近5日（最高-最低）÷ 均价 **≤ 5%** | 窄幅运行 |
| 横盘持续时间 | **≥ 5个交易日**，理想8~15日 | 越长越充分 |
| 重心是否下移 | 近5日均价 vs 近10日均价，偏差 **≤ 1%** | 没下移=真横盘，下移=阴跌 |
| 支撑位守住 | 收盘价始终在某关键支撑位（MA60或前低）上方 | 有底托着 |

**达标判断**：
- 4项全满足 → ✅ 横盘确认
- 满足2~3项 → ⚠️ 基本横盘，可继续观察
- 不满足 → ❌ 还在趋势运行中

**怎么区分「横盘」和「阴跌」**：
- 横盘：价格区间固定，重心不下移，缩量
- 阴跌：每天创新低，重心持续下移，即使缩量也不算企稳


### 指标③：均线粘合 ⭐⭐⭐⭐

> **核心逻辑**：均线粘合说明短中期成本趋于一致，是方向选择的前兆。
> 一旦放量向上突破粘合区，通常会有一段加速行情。

| 子项 | 达标参数 | 说明 |
|------|----------|------|
| MA5/MA10/MA20间距 | 三线最大值与最小值之差 ÷ 均值 **≤ 3%** | 三线粘合 |
| 现价与MA60 | 现价 > MA60，且MA60斜率向上或走平 | 中期趋势不坏 |
| MA5与MA10 | MA5即将金叉MA10（或刚金叉） | 短期拐头向上 |

**达标判断**：
- 3项全满足 → ✅ 均线粘合待发
- 满足2项 → ⚠️ 接近粘合
- 不满足 → ❌ 均线发散，时机未到


### 指标④：K线实体缩小 ⭐⭐⭐

> **核心逻辑**：K线实体越来越小 = 每天的涨跌幅越来越小 = 多空力量趋于平衡 = 跌不动了。

| 子项 | 达标参数 | 说明 |
|------|----------|------|
| 近5日实体均值 / 近20日实体均值 | **≤ 0.5** | 实体缩至一半以下 |
| 近3日最大日振幅 | **≤ 3%** | 波动很小 |
| 近5日涨跌幅绝对值均值 | **≤ 1.5%** | 每天涨跌都很微小 |

**达标判断**：
- 3项满足2项以上 → ✅ 实体缩小确认
- 满足1项 → ⚠️ 有迹象
- 不满足 → ❌ 波动仍大


### 指标⑤：下影线阳线 ⭐⭐⭐

> **核心逻辑**：盘中被打压下去，但尾盘有资金拉回来收红。
> 说明下方有人在主动承接，不让股价跌下去——这就是主力的托底行为。

| 子项 | 达标参数 | 说明 |
|------|----------|------|
| 收盘 > 开盘 | 必须收红 | 基本条件 |
| 下影线长度 ≥ 实体2倍 | (开盘-最低) ≥ 2×(收盘-开盘) | 长下影=有人托 |
| 涨跌幅 | 0% ~ +2% | 微涨，不是大阳线 |
| 出现频率 | 近5日内至少1根 | 有即加分 |

**达标判断**：
- 近5日出现1根以上 → ✅ 有托底信号
- 未出现 → ⚠️ 不加分也不减分（此项为加分项）


### 指标⑥：小十字星 ⭐⭐

> **核心逻辑**：开盘价≈收盘价，说明一天下来多空打了个平手，
> 在底部区域出现代表卖方力量衰竭，变盘在即。

| 子项 | 达标参数 | 说明 |
|------|----------|------|
| 实体幅度 | \|收盘-开盘\| ÷ 开盘 **≤ 0.5%** | 几乎没涨没跌 |
| 有上/下影线 | 影线长度 > 实体2倍 | 盘中有波动但收回来了 |
| 出现频率 | 近5日内至少2根 | 反复出现才有意义 |

**达标判断**：
- 近5日出现2根以上 → ✅ 犹豫/平衡信号确认
- 出现1根 → ⚠️ 有迹象
- 没有 → 不加分


### 指标⑦：红绿交替 ⭐⭐

> **核心逻辑**：底部不再单边下跌，而是涨一天跌一天，
> 说明有资金在逢低吸纳，也有短线在高抛低吸，筹码在换手。

| 子项 | 达标参数 | 说明 |
|------|----------|------|
| 近5日K线颜色 | 红绿交替（不连续出现3根同色） | 有来有回 |
| 近5日每日涨跌幅 | 每日在 -2% ~ +2% 之间 | 波动温和 |
| 近5日累计涨跌 | -2% ~ +3% | 5天下来几乎走平 |

**达标判断**：
- 3项全满足 → ✅ 红绿交替确认
- 满足1~2项 → ⚠️ 部分符合
- 不满足 → ❌

## 第三关：加分项与排除项


### 加分项（有则更好，没有不影响）

| 编号 | 加分条件 | 参数 | 加分原因 |
|:---:|------|------|----------|
| B1 | 洗盘未破MA60 | 回调最低价 > MA60 | 主力护盘意愿强 |
| B2 | 底部突现单日放量阳线 | 某日量比 > 2.0 且涨幅 > 3% | 可能是主力试盘信号 |
| B3 | 有基本面催化 | 季报预告、行业政策、重组等 | 有题材驱动 |
| B4 | 机构持仓上升 | 近一季基金持仓比上升 | 机构认可 |
| B5 | 板块正在轮动 | 所属板块近5日涨幅前20% | 借板块势启动 |


### 排除项（踩中任一条，直接放弃）

| 编号 | 排除条件 | 为什么 |
|:---:|------|--------|
| X1 | 近5日出现过单日跌幅 > 5% | 还在杀跌中，不是洗盘尾声 |
| X2 | 近20日连续下跌超过10天 | 单边下跌趋势未止 |
| X3 | 换手率突然飙升 > 8%（大盘股 > 5%） | 可能是主力出货 |
| X4 | MA60向下且现价 < MA60 | 中期趋势向下 |
| X5 | 近半年有重大利空（暴雷、处罚） | 基本面风险 |

## 第五关：等待启动信号（选出来后何时买？）

> **重要原则**：选出来不等于马上买！必须等启动信号出现才动手。

### 启动信号（出现任一即可）

| 信号 | 具体表现 | 如何判断 |
|------|----------|----------|
| **放量突破** | 某日成交量 ≥ 近5日均量的 **2倍**，且收涨 > 3% | 量是关键，价是确认 |
| **突破横盘上沿** | 收盘价突破近期横盘区间的最高价 | 必须是收盘价突破，盘中不算 |
| **MA5金叉MA10** | 5日均线从下方穿越10日均线 | 短期趋势拐头 |
| **集合竞价异动** | 早盘竞价高开 > 1% 且竞价量能放大 | 可辅助判断当日是否会启动 |


### 两种标准买入模式

#### 模式A：放量突破后回踩买入（更安全）

```
第一步：确认突破
  → 某日收盘价站上关键压力位（如MA20、前高、横盘上沿）
  → 当日成交量 > 近20日均量 × 1.3倍
  → 当日换手率明显放大

第二步：等待回踩（1~5天）
  → 突破后股价从高点小幅回落
  → 回踩过程中成交量萎缩（量比 < 0.8）

第三步：确认支撑后买入
  → 回踩过程中收盘价不跌破关键支撑（如突破位）
  → 出现缩量+跌幅收窄+收红 → 次日买入

❌ 失败判断：回踩跌破突破位 → 假突破，放弃
```

#### 模式B：缩量企稳低吸（需耐心）

```
第一步：价格进入目标区间
  → 股价回调到预设的支撑区间（如MA20/MA60附近）

第二步：观察企稳信号（至少看3天，理想5~8天）
  → 连续3天+收盘价维持在区间内
  → 成交量逐日萎缩，量比 ≤ 0.7
  → 日振幅收窄至3%以内
  → 出现小十字星或下影线阳线

第三步：确认不破底线后买入
  → 收盘价始终不破下方止损位
  → 连续缩量后某日收红 → 当日或次日买入

❌ 失败判断：收盘跌破止损位 → 企稳失败，放弃
```

### 仓位管理

| 阶段 | 仓位 | 条件 |
|------|:---:|------|
| 首次建仓 | **半仓** | 出现启动信号或企稳确认时 |
| 加仓 | **加至8成** | 突破确认（站稳3天不回落） |
| 满仓 | 不建议 | 单只股票永远不满仓 |

## 附录：常见问答

**Q：缩量了但还在跌，算企稳吗？**
A：不算。缩量是必要条件但不是充分条件。必须同时满足「缩量 + 跌幅收窄/走平 + 不破支撑」三个条件才算企稳。单纯缩量下跌叫「阴跌」，可能还没到底。

**Q：盘中跌破支撑位但收盘拉回来了，算破位吗？**
A：以收盘价为准。盘中跌破但收盘拉回 → 不算破位，反而说明有资金托底（下影线信号）。但如果连续多日盘中跌破，即使收盘拉回也要警惕。

**Q：突破当天能不能直接追？**
A：不建议。突破当天往往涨幅已大（3%~涨停），追高成本高、风险大。更好的做法是等1~3天回踩确认后再介入，虽然少赚一点但安全得多。

**Q：横盘了很久但就是不启动，要等多久？**
A：一般横盘5~15天是正常洗盘周期。超过20天仍无启动信号，可能是：①主力控盘但在等催化剂（板块轮动/消息面）；②主力已放弃。超过20天可降低关注优先级，但不必完全排除。

**Q：满足A级评分但没有启动信号，能提前买吗？**
A：可以小仓位（2~3成）提前埋伏，但必须设好止损。正式加仓必须等启动信号。没有信号就重仓是赌博不是投资。

**Q：怎么设止损？**
A：两个原则：①跌破企稳区间的下沿（如回调企稳在42~43，止损设在41）；②跌破MA60。两者取较近的那个。一般止损幅度控制在买入价的5~8%以内。
